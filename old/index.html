How to read the dashboard

Each indicator is a single monthly series shown against grey bands marking the twelve Canadian recessions identified by the CD Howe Institute Business Cycle Council. The traffic-light colour at the top-left of each card is a simple rule-based signal — useful as a visual shorthand, not a substitute for judgement. No single indicator should drive a recession call; patterns across several matter more.

Signal rules. Yield curve: red when inverted (spread < 0). Unemployment: red if the Sahm rule is triggered (3-month average ≥ 0.5pp above its trailing 12-month low). Housing starts: red if the 3-month average is below the 12-month average by >10%. Retail sales YoY: red if negative, yellow below 1%. BCPI YoY: red below −20%. CPI-trim: neutral (context rather than a recession signal — the 1–3% BoC target band is shaded).

Data sources

Bank of Canada Valet API for the BoC Commodity Price Index and CPI-trim / CPI-median. Statistics Canada Web Data Service for Labour Force Survey unemployment (14-10-0287), CMHC housing starts (34-10-0158), retail trade (20-10-0008 + 20-10-0056 spliced at 2018-12), headline CPI (18-10-0004), and government bond yields (10-10-0122). Data refreshes nightly via GitHub Actions.
